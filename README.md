# ContactsAppUsingIsar

Flutterの[IsarDB](https://github.com/isar/isar)を使用した連絡先管理アプリを作成しています。

# 使用している技術や構成
- Riverpod(StateNotifier)
- IsarDB
- DDD(レイヤードアーキテクチャ)
